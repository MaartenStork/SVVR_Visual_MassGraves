Body 4 - Dataset (2021.020)
============================

This folder should contain the DICOM data for Body 4.

Expected contents:
------------------
- DICOM/          → Raw DICOM image files
- DICOMDIR        → DICOM directory index file
- BIN/            → Viewer binaries (optional)
- HELP/           → Help documentation (optional)
- INFO/           → Additional info files (optional)
- ABOUT.HTM       → About page
- AUTORUN.INF     → Autorun config
- DISCINFO.XML    → Disc info metadata
- MOREINFO.HTM    → Additional info page
- STUDYMAP.TXT    → Study mapping
- Slices/         → Extracted slice images (optional)
- *.blend         → Blender project files (optional)
- *.nrrd          → Segmentation files (optional)

The DICOM/ folder is required for the viewer to work.
Place the original CD/DVD contents here.
